HTM.parseinterp ‚Äî Method
parseinterp(io::IO)
parseinterp(fallback, io::IO)

Parse an interpolation as string, including $.

The input must start with $ if no fallback function is given. The fallback function is passed to readuntil if the input does not start with $.

julia> HTM.parseinterp(IOBuffer(raw"$((1, (2, 3)))..."))
"\$((1, (2, 3)))"
